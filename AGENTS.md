# AGENTS.md — FGC Studio

## GSD Workflow Enforcement

This project uses the **Get Shit Done (GSD)** workflow.

### Commands
- `/gsd-new-project` — Initialize a new project
- `/gsd-map-codebase` — Analyze existing codebase
- `/gsd-discuss-phase <N>` — Gather context before planning a phase
- `/gsd-plan-phase <N>` — Plan a specific phase
- `/gsd-execute-phase <N>` — Execute a planned phase
- `/gsd-transition` — Complete a phase transition
- `/gsd-complete-milestone` — Complete a milestone
- `/gsd-progress` — Show project progress
